# Хрусталева — запросы 1С и СКД

Справочник по **Хрусталева Е.Ю. «Разработка сложных отчётов в „1С:Предприятии 8“. Система компоновки данных»** (изд. 2).

Книга написана про отчёты СКД, но примеры задают **общие паттерны языка запросов**: соединения, виртуальные таблицы, параметры, периоды, остатки — они применимы в **любом** коде 1С.

## Когда подключать этот скил

**Язык запросов (модули, консоль, запросы в коде):**

- LEFT / INNER, несколько таблиц в одном запросе;
- виртуальные таблицы регистров (`Остатки`, `ОстаткиИОбороты`, `СрезПоследних`);
- параметры `&Период`, `В (&Ссылка)`;
- отбор в `ГДЕ` vs группировка;
- остатки с регистратором и полями периода.

**СКД / отчёты:**

- наборы данных, связи, объединение;
- параметр связи в запросе приёмника;
- итоги по полям остатка, роли полей.

**Не дублировать:** синтаксис `ВЫБРАТЬ`, `ВЫРАЗИТЬ`, RLS, индексы — см. `composing-1c-queries` и `query-optimization-tips`.

## Алгоритм

1. Запрос **в модуле / консоли** → начать с **`08-query-language-general.md`**.
2. **СКД / схема отчёта** → `01`, `03`, `04` + при необходимости `08`.
3. Правка текста запроса в наборе данных → `02` или `03` + `05`.
4. Неверные остатки → `06`.
5. Перед сдачей → `07`.

## Ключевые правила

**Везде (запросы):** ВТ — условия в параметрах; LEFT — весь левый набор; `&Параметры`; осмысленные `КАК`.

**СКД:** связь ≈ LEFT; объединение ≈ UNION; приёмник — `В (&...)`.
