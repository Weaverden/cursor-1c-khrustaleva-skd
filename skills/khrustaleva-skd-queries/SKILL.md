# Хрусталева — сложные отчёты и запросы в СКД

Справочник по книге **Хрусталева Е.Ю. «Разработка сложных отчётов в „1С:Предприятии 8“. Система компоновки данных»** (изд. 2).

Структурированные выжимки — в `references/` этого скила (устанавливаются вместе с `cursor-1c-skills`).

Опционально: полный PDF и текст с номерами страниц — см. `references/SOURCE.md` и скрипт `scripts/download-book.ps1` (Windows) / `scripts/download-book.sh` (macOS/Linux).

## Когда подключать этот скил

- отчёт на **системе компоновки данных** (СКД / DCS);
- несколько **наборов данных** или выбор: один запрос vs связи vs объединение;
- **связь** регистра с документом, параметр `В (&Ссылка)`;
- **остатки** в отчёте, виртуальные таблицы, итоги по полям остатка;
- неверные итоги / «плывут» остатки при детализации по регистратору.

Для **обычных запросов вне СКД** (модули, консоль) — сначала `composing-1c-queries`; этот скил — про отчёты и компоновку.

## Ключевые правила (кратко)

- **Связь наборов** ≈ LEFT JOIN; **Объединение** ≈ UNION ALL между наборами.
- В запросе **приёмника** всегда `ГДЕ ... В (&ПараметрСвязи)` при связи с родителем.
- **Отбор** жёсткий — в запросе; **группировки** — в настройках СКД.
- Для итогов по **остаткам**: в запросе цепочка полей-периодов (`ПериодСекунда` + `Регистратор`).
